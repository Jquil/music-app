# 前言

因为版权问题，所以在听歌的时候总是需要切换APP，麻烦

并且，因为穷，听不起付费歌曲

所以就萌生了自己动手开发一个音乐APP的念头

因为我想弄个跨平台的APP，在一番考量后决定用Flutter作为跨平台框架


# 实现

1. 搜集数据接口
2. 定下基本框架
3. 设计数据库
4. 思考操作流程

然后就可以开始搬砖了~
